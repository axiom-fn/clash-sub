# 🎯 CLASH-SUB

> Clash 訂閱轉換工具

---

## ✨ 特色

- 🚀 **vless+reality 完整支援**
- 📦 **vmess:// 解析**
- 🔧 **規則集自動合併**
- 🎭 **佔位符節點**
- ✏️ **節點改名**
- ⚡ **並行抓取**

---

## 📲 一鍵安裝

**VPS 或 Termux**：

```bash
curl -sL https://raw.githubusercontent.com/axiom-fn/get/main/install.sh | bash
```

安裝後輸入：

```bash
clash-sub
```

進入操作台。

---

## 🎮 使用流程

1. 啟動後端（選項 8）
2. 新增訂閱配置（選項 1）
3. 設定規則集 / 運行指令 / 佔位符
4. 生成訂閱 URL（選項 7）
5. 複製 URL 到 Clash 訂閱

---

## 🛠️ 環境

- Python 3.7+
- Linux / Termux
- Flask + Requests + PyYAML
